WarGame — suninatas

웹 해킹·시스템 해킹·리버싱·포렌식·암호학 등 다양한 카테고리를 제공하는 국내 CTF 스타일 워게임 플랫폼 풀이 기록

Challenges

1

Replace()(치환), MID(문자열, 시작위치, 길이) 함수 코드 분석 → 코드 로직에 맞는 값을 입력해 key 획득.

2

로그인 폼에서 id/pw가 같아야 한다는 힌트가 있지만, 자바스크립트가 id=pw일 때 값을 공백 처리하는 함정이 걸려있음 → 개발자 도구 콘솔에서 document.web02.submit();을 직접 호출해 강제 제출로 우회.

3

notice 게시판은 글쓰기 기능이 없고 free 게시판에만 가능한 구조. URL 경로를 /board/free/write에서 /board/notice/write로 직접 조작해 원래 막혀있던 게시판에 글쓰기 성공.

4

Point 값을 26까지 올리면 팝업이 뜨는 구조, 소스코드에서 User-Agent 관련 힌트 발견 → 프록시로 User-Agent와 point 값을 함께 조작해 해결.

5

스크립트가 값을 공백 처리하는 난독화된 코드 확인(eval() 사용). jsnice.org에서 난독화 해제 후 로직을 분석해 해결.

6

SQL 쿼리문 확인 결과 =이 공백 처리되는 필터가 있는 것으로 추정 → LIKE 구문으로 대체해 필터 우회.
