webgoat, WAP

OWASP WebGoat 실습과, Apache+ModSecurity(WAF) 직접 구축·룰 작성 실습 기록

1. WebGoat 실습

Bypass a Path Based Access Control Scheme

서버가 사용자 실제 권한이 아니라 URL 경로 문자열만으로 접근을 허용하는 취약한 구현. ../로 최상위 루트까지 이동해 /etc/shadow 같은 민감 파일에 접근, 획득한 해시는 john으로 크래킹.

DOM-Based XSS (AJAX Security)

자바스크립트가 사용자 입력값을 검증 없이 DOM에 직접 삽입할 때 발생. <img src=x onerror="document.body.innerHTML='...'"> 형태로 화면 전체를 덮어써서 사용자가 정상적으로 이용하지 못하게 만드는 것까지 실습.

Multi Level Login 2 (2차 인증 우회)

2차 인증 단계에서 사용자 이름만으로 검증하는 구조. 다른 사용자 ID(Jane)로 바꿔 요청하면 해당 사용자의 신용카드 정보가 그대로 노출됨을 확인.

Discover Clues in the HTML

페이지 소스코드에 로그인 계정 정보가 그대로 노출되어 있어, 이를 이용해 로그인 성공.

Phishing with XSS

XSS로 정상 사이트 안에 가짜 로그인 폼을 삽입해 사용자 자격 증명을 탈취하는 공격. 공격자 서버에 hacking.html을 띄우고, iframe으로 삽입하거나 document.body.innerHTML을 가짜 로그인 폼으로 통째로 교체하는 방식으로 실습.

Command Injection

파일 내용을 읽어 출력하는 기능에서, ;로 명령어를 추가 삽입해 원래 명령어와 함께 실행되는 것을 응답 결과로 확인.

Numeric / String SQL Injection

숫자형 파라미터는 따옴표 없이 바로 SQL 문법 삽입(1 OR 1=1), 문자열형 파라미터는 따옴표를 맞춰서 삽입(' OR '1'='1, Smith' or 1=1 --)하는 두 유형을 비교 실습해 WHERE 조건을 항상 참으로 만들어 전체 데이터 노출.

Database Backdoors

SQL Injection으로 데이터를 임의로 변경하는 것까지 실습 — 101; update employee set salary=100000 where userid=101 형태로 급여 값 변경 성공.

2. WAF (ModSecurity) 구축 실습

ModSecurity는 SQL Injection, XSS 같은 공격 패턴을 실시간으로 탐지·차단하는 오픈소스 웹 애플리케이션 방화벽(웹서버 모듈 형태).

방화벽 / IPS·IDS / WAF 비교

방화벽IPS/IDSWAF
감시 기준IP/포트알려진 공격 시그니처웹 요청 내용(URL, 파라미터)
대상 범위모든 트래픽모든 프로토콜웹(HTTP/HTTPS)만
이해도얕음중간깊음(웹앱 로직까지)

설치 및 기본 설정

sudo apt install -y apache2 libapache2-mod-security2
sudo a2enmod security2
sudo systemctl start apache2

sudo cp /etc/modsecurity/modsecurity.conf-recommended /etc/modsecurity/modsecurity.conf
vi /etc/modsecurity/modsecurity.conf
SecRuleEngine On          // 감지만 하던 기본값을 차단까지 되도록 변경
SecAuditLogParts ABCEFHIJZ

SecRule 문법

SecRule 변수 "패턴" "액션들"

변수(어디를 검사할지: ARGS, REQUEST_URI, REQUEST_BODY, REQUEST_COOKIES 등) → 패턴(무엇을 찾을지: @contains, @rx 정규식, @eq 등) → 액션(발견 시 무엇을 할지: id, phase, deny, log, status 등) 3단계 구조.

Phase시점
1요청 헤더 수신 직후 (URL, 헤더, 쿠키 — 빠른 차단용)
2요청 본문까지 수신 완료 (POST 데이터 — 가장 흔히 사용)
3응답 헤더 생성 후
4응답 본문 생성 후 (정보 유출 탐지 등)
