진입 시 자바스크립트가 실행되며 강제로 메인 페이지로 리다이렉트. 브라우저 자바스크립트 비활성화 또는 Burp Suite로 응답 패킷의 이동 코드를 삭제해 우회 → getFlag 링크가 노출되어 클릭 후 해결.

$decode_id = str_replace("!","1",...)  // 특수문자→숫자 치환
for($i=0;$i<20;$i++){ $decode_id = base64_decode($decode_id); }
if($decode_id=="admin" && $decode_pw=="nimda"){ solve(6); }

쿠키 값이 숫자→특수문자 치환 후 20번 Base64 인코딩된 형태로 저장되고, 디코딩 시 admin/nimda가 나와야 통과되는 구조. 디코딩 로직을 반대로 실행하는 인코딩 스크립트를 작성해 admin/nimda를 20회 인코딩한 값을 쿠키로 덮어써서 통과.

old-42

특정 확장자는 다운로드가 막혀있음. 개발자 도구로 ?down= 파라미터 값이 Base64 인코딩된 파일명인 것을 확인 → 원하는 파일명(flag.docx)을 동일하게 인코딩해서 파라미터로 전달, 다운로드 성공.

SQL Injection 문제

old-56

관리자 글 본문은 직접 접근이 막혀있지만, 검색 결과가 본문 내용을 기준으로 노출되는 것을 확인(예: flag{ 검색 시 admin 글만 노출). 파이썬 스크립트로 Blind SQL Injection을 자동화해 문자 단위로 값을 추출.
