## 1. 네트워크 기초

### OSI 3계층 vs 4계층 차이

### 핵심 개념

- 3계층: 어디로 보낼지 결정 (경로 선택)
- 4계층: 어떻게 보낼지 결정 (전송 방식)

| 구분 | 3계층 (네트워크) | 4계층 (전송) |
| --- | --- | --- |
| 역할 | 경로 선택 | 전송 방식 |
| 주소 | IP 주소 | 포트 번호 |
| 장비 | 라우터 | 없음 (논리적) |
| 프로토콜 | IP | TCP / UDP |

### 사설 IP 대역

- 10.0.0.0 ~ 10.255.255.255
- 172.16.0.0 ~ 172.31.255.255
- 192.168.0.0 ~ 192.168.255.255

→ 인터넷 충돌 막으려고 따로 빼둔 내부 전용 주소

→ 192.168 대역은 범위가 상대적으로 작아서 자주 사용

### IPv4 / IPv6 / MAC 주소 비교

| 구분 | IPv4 | IPv6 | MAC |
| --- | --- | --- | --- |
| 비트 | 32bit | 128bit | 48bit |
| 묶음 | 4개 | 8개 | 6개 |
| 형식 | 10진수(1~3자리 × 4묶음) | 16진수(4자리 × 8묶음) | 16진수(2자리 × 6묶음) |
| 구분자 | . | : | : / - |

## 2. 패킷 트레이서

패킷 트레이서는 Cisco 장비를 기반으로 한 **가상 환경**에서 네트워크를 구성하고 **실습**할 수 있는 시뮬레이션 소프트웨어이다.

### 기초 명령어

### 장비 이름 변경

!image.png

- en > conf t 로 설정 모드로 들어간다
- hostname [이름] 명령어로 Router > R-1 로변경한다

### 모드 이동

!image.png

- 인터페이스 모드에서 end 명령어를 입력하니 관리자 모드로 이동 됐다.

!image.png

- exit 는 한단계 전으로 이동

### 비밀번호 설정

!image.png

- 관리자 모드의 비밀번호를 1234로 설정

!image.png

- 관리자 모드 진입 시 위에서 설정한 비밀번호 필요

!image.png

- show run 명령어로 현재 설정을 보면 비밀번호가 평문으로 그대로 노출

- 다시 show run 명령어로 현재 설정을 보면 비밀번호가 암호화 되어 있음
