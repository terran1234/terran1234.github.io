리눅스 기본 명령어, SQL

패키지 설치·제어, 파일/권한 명령어, 간단한 웹서버 구성, DB 접속과 SQL DDL/DML 기본 문법 정리

1. 리눅스 기본 명령어

프로그램 설치 & 제어

yum update -y
dnf install -y httpd php mariadb-server

yum은 패키지 관리 도구, dnf는 더 개선된 yum(Dandified YUM). httpd=아파치 웹서버, php=웹 개발 언어, mariadb-server=DB 서버. HTTP는 프로토콜 이름, httpd는 프로그램 이름.

systemctl status [서비스 이름]
systemctl start [서비스 이름]
systemctl restart [서비스 이름]

status: active(running)=실행 중, inactive=꺼짐, failed=오류.

디렉토리 / 복사

pwd
ls -al
cd
mkdir [파일명]

cp 원본파일 대상파일
cp -r dir1 dir2   // 디렉토리는 -r 필수

파일 관련 & 권한

touch test.txt
cat test.txt
head -5 test.txt
tail -10 test.txt
less test.txt

cat=이어붙여 출력, head/tail=위/아래 지정 줄만 출력, less=페이지 단위로 천천히(q로 종료).

권한: d/-/l = 디렉토리/일반파일/링크. r(4)/w(2)/x(1), 사용자(User)/그룹(Group)/기타(Other) 순서.

chmod 755 test.txt

계정 생성 / VI 편집기

useradd [계정이름]
passwd [비밀번호]
  • i — 입력 모드 진입
  • esc — 명령 모드로 전환
  • :wq! — 저장 후 강제 종료
  • :set nu — 라인 숫자 표시

방화벽 제어

systemctl status firewalld
firewall-cmd --permanent --add-port=80/tcp
firewall-cmd --reload
firewall-cmd --list-all

--permanent은 영구 적용(재부팅해도 유지)이지만 파일에만 저장되므로, --reload로 다시 불러와 실제 적용해야 함.

2. 웹 서버 (PHP 테스트)

vi /var/www/html/phptest.php
<?php
phpinfo();
?>

/var/www/html이 웹페이지 저장소. http://[IP]/phptest.php로 접속해 확인.

3. DB (MariaDB)

systemctl start mariadb
mysql_secure_installation

여기서 물어보는 root password는 리눅스 root가 아니라 DB의 root 비밀번호.

mysql -u root -p

원격 접속 시 방화벽에 3306 포트 추가 필요. (none)은 아직 DB가 선택되지 않은 상태.

4. SQL

DDL — CREATE / ALTER / DROP

SHOW DATABASES;
CREATE DATABASE [DB명];
USE [DB 이름];

CREATE TABLE student ( Date int(4), Phone int(10) );
SHOW TABLES;
DESC student;
ALTER TABLE student ADD Age1 INT(3);
ALTER TABLE student ADD Address VARCHAR(10) AFTER Date;
ALTER TABLE student ADD MBTI CHAR(4) FIRST;
ALTER TABLE student MODIFY Name VARCHAR(10);
ALTER TABLE student DROP Age1;

AFTER=특정 필드 뒤에 추가, FIRST=맨 앞에 추가, MODIFY=자료형/위치 수정, DROP=필드 제거.

DROP DATABASE [DB명];
DROP TABLE [테이블명];

DML — INSERT / SELECT / UPDATE / DELETE

INSERT INTO student VALUES ('Park', 2026, 'Seoul', 12345678, 32);
INSERT INTO student (Name, Address) VALUES ('Lee', 'Busan');

SELECT * FROM student;
SELECT Name, Address FROM student;

UPDATE student SET address = 'Daegu' WHERE Name = 'Lee';

DELETE FROM student WHERE Name = 'Hong';

WHERE 절 없이 UPDATE/DELETE 실행하면 전체 행이 대상이 되므로 주의.
